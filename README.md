# 🚀 Krishna Singh - Developer Portfolio

A modern, responsive, and interactive developer portfolio built using **HTML, CSS, and JavaScript** to showcase my projects, technical skills, achievements, and contact information.

## 🌐 Live Demo

🔗 https://krishnasingh-protfolio.netlify.app/

---

## 📌 Features

- Responsive Design
- Dark / Light Theme Toggle
- Animated Hero Section
- Typing Animation
- Scroll Reveal Animations
- Skill Progress Bars
- Interactive Project Cards
- Contact Form using EmailJS
- Smooth Scrolling Navigation
- Mobile Friendly Navigation
- Scroll-to-Top Button
- Modern UI with Glassmorphism Effects

## 📬 Contact Form

The portfolio uses **EmailJS** to send messages directly from the frontend.

When a visitor submits the form:

- Name
- Email
- Subject
- Message

The message is delivered to my email inbox without requiring a backend server.
